1.Выбрать камеру и ввести разрешение камеры (не большое, до 1024 на 768).
2.Выбрать монитор и разрешение.
3.Добавить картинку, которую нужно найти, через  Файл-открыть снимок.
4. Запустить камеру
5. Если сразу не найдется объект в камере, может вылететь по эксепшину
6. При запуске будет три окна: одно  окно приложения (со стримом), во втором показана маска найденного объекта,
в третьем - эта же маска (в которой должна быть шахматка), но растянута по величина экрана (пока просто по величине виджета)
